# YuniqiGPT
A cpenAI GPT PC client
基于openAI API 开发的电脑客户端,主要为自已练手，语言为python,GUI用的是DearPY GUI 1.8,实现了网页版chatGPT的主要功能,上下文相关,流式传输之类.还可以自定义详细的参数以及系统提示语
